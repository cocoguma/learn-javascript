# 1、基础组件

  1. View：等同于div，特色属性有按下弹起等样式可以修改。阻止冒泡hover-stop-propagation(大部分不支持，微信小程序支持)
  2. Text：等同于span。特色的属性是，是否可选selectable、user-select(wx)、是否允许连续空格space
  3. scroll-view：等同于设置了scroll的div。
  4. swiper：轮播图。
  5. image：img

# 2、路由组件

navigator：类似于a标签 仅设置url的话，默认不会销毁原页面式的跳转，返回可以回到原页面。 若想销毁原页面。open-type 设置为reLaunch target属性可以跳转其他小程序

# 3、页面生命周期

文档：教程-页面-页面生命周期

  1. onLoad：在vue的生命周期之前，用来页面之间路由传参。
  2. onReady：挂载完毕，类似onMounted
  3. onShow：第一次进入页面时，在load和ready之间。后续返回页面，都会执行。
  4. onHide：和show是一对，离开当前页面时触发。
  5. onUnload：页面卸载，有点像destroy。
  6. onPageScroll：页面滚动

# 4、小程序尺寸单位rpx

基于750px缩放。如果是750px的稿就是1:1
